# City Power Washing

Modern, SEO-optimized pressure washing website for Central Florida.

## Stack

- **Next.js 16** (App Router, static generation)
- **Tailwind CSS 4**
- **HTML5 Canvas** water ripple hero effect
- **JSON-LD** structured data (LocalBusiness, Service, FAQ, Breadcrumb)

## Pages (19)

Home, 8 residential services, 2 commercial services, 6 city landing pages, About, Free Quote.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy (Vercel)

```bash
npx vercel --prod
```

## SEO

- Per-page titles, meta descriptions, canonical URLs
- `sitemap.xml` and `robots.txt` auto-generated
- Schema markup on every service and city page
